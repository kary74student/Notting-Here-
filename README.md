# Notting-Here-
Repositorio de git para el proyecto Notting Here. 

Proyecto: Notting Here

Pitch: 
Notting Here es un juego de suspenso donde el jugador despierta atrapado en un suburbio abandonado, la única forma de escapar es explorando el mapa y encontrando tres llaves ocultas, un monstruo recorre la zona y reaccionará a la presencia del jugador, la tensión aumenta a medida que el jugador administra su resistencia al correr, cada decisión puede ser la diferencia entre escapar o perder.

Mecánicas: 
- Movimiento libre e ilimitado dentro del mapa.
- Correr por un tiempo limitado usando resistencia.
- Exploración del entorno para encontrar objetos clave.
- Inventario para visualizar las llaves recolectadas.
- Sistema de derrota al ser atrapado por el enemigo.

Controles:
  Teclado
    W / A / S / D – Movimiento
    Shift – Correr
    E – Interactuar
    I – Abrir inventario
    Esc – Pausa
  Gamepad
    Joystick izquierdo – Movimiento
    Botón A / X – Interactuar
    Botón B / Círculo – Correr
    Botón Y / Triángulo – Inventario
    Start - Pausa

Objetivo de Rendimiento: 60fps
  60FPS estables
  CPU: Gama media (Ryzen 5 / Intel i5 o equivalente)
  GPU: GTX 1650 / RX 570 o superior
  RAM: 8 GB

Métricas que van a reportar: FPS, load time, build size, resolución de texturas.
  FPS promedio y mínimo.

  Tamaño final del build.
